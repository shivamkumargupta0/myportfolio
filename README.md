Project Description

This is a static portfolio website developed using HTML and CSS.
It showcases my personal introduction, skills, projects, and contact information.
The website is created as part of a practical assignment to demonstrate web development skills and usage of development tools such as VS Code, Git, and GitHub.
The project is hosted on GitHub and maintained with proper version control best practices and documentation.

Installation and Usage Instructions
To Run Locally:

Clone the repository

 git clone https://github.com/shivamkumargupta0/myportfolio.git


Open the folder in any code editor such as VS Code.
